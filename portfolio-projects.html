Earthquake Data Analysis

Domain/Function: Environmental Data Science | Geospatial Analytics

Analyzed global earthquake data using Python to uncover patterns in magnitude, frequency, and location. Applied data visualization and statistical techniques to support risk assessment and geospatial insights for disaster preparedness.

Project Details

In this project, I explored global earthquake data to uncover patterns in seismic activity using Python libraries such as Pandas, Matplotlib, and Seaborn. I analyzed trends in magnitude, depth, and frequency over time, and visualized earthquake distributions across geographic regions. By examining correlations between depth and magnitude, I derived insights that can support disaster preparedness and risk assessment. This project not only strengthened my skills in data cleaning, visualization, and exploratory data analysis but also deepened my understanding of how data science can be applied to real-world environmental challenges. It reflects my commitment to using data as a tool for driving meaningful, human-centered impact.
